# Sensor Review Checklist

- [ ] Timestamp present or explicitly missing.
- [ ] Platform location present or explicitly missing.
- [ ] Platform velocity present or explicitly missing.
- [ ] Sensor type identified.
- [ ] Sensor mode identified.
- [ ] Focal length or field of view present or explicitly missing.
- [ ] Frame rate present or explicitly missing.
- [ ] Target range measured, bounded, assumed, or missing.
- [ ] Azimuth/elevation present or explicitly missing.
- [ ] Weather and wind context reviewed.
- [ ] Radar correlation reviewed.
- [ ] Infrared correlation reviewed.
- [ ] Optical correlation reviewed.
- [ ] Chain of custody reviewed.
- [ ] Compression history reviewed.

No propulsion conclusion should be made from video alone.
