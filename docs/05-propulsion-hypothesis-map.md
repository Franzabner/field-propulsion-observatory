# Propulsion Hypothesis Map

Hypotheses are review categories, not conclusions. They help identify what evidence would be required before stronger language is allowed.

## Known / Conventional

| Category | Screening questions |
| --- | --- |
| Aircraft | Is there a flight path, lighting pattern, acoustic signature, speed envelope, or radar correlation? |
| Drone | Is range short enough, motion compatible, and acoustic/visual behavior consistent? |
| Balloon | Do wind, altitude, drift, and apparent hovering fit? |
| Satellite | Does timing, elevation, angular speed, and illumination fit orbital pass behavior? |
| Bird/insect | Does near-field range explain apparent speed or size? |
| Atmospheric effect | Are clouds, mirage, reflection, plasma, lightning, or weather plausible? |
| Sensor artifact | Do optics, compression, stabilization, gain, or processing explain the track? |
| Electronic warfare / spoofing | Does the record depend on radar/electronic tracks vulnerable to deception or clutter? |
| Classified aerospace platform | Is the behavior within plausible advanced but conventional systems? Public evidence may still be insufficient. |

## Speculative Engineering

| Category | Boundary language |
| --- | --- |
| Plasma sheath / flow control | Not demonstrated as a public propulsion technology. Requires aerodynamic, thermal, and power evidence. |
| Electromagnetic field coupling | Not demonstrated as a public propulsion technology. Requires calibrated field, power, and force evidence. |
| Magnetohydrodynamic interaction | Not demonstrated as a public propulsion technology. Requires medium interaction, power, heat, and control evidence. |
| Inertia-management hypothesis | Not demonstrated as a public propulsion technology. Requires acceleration, occupant/load, power, and conservation analysis. |
| Gravitational / spacetime metric hypothesis | Not demonstrated as a public propulsion technology. Requires extraordinary calibrated evidence and independent review. |
| Vacuum-energy / zero-point hypothesis | Not demonstrated as a public propulsion technology. Requires evidence far beyond public video or anecdote. |

## Rule

Speculative vocabulary can be used to define evidence requirements. It must not be used as a conclusion from unresolved observations.
