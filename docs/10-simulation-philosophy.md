# Simulation Philosophy

Simulation is a discipline of declared assumptions.

The first goal is not to prove exotic propulsion. The first goal is to prove the reasoning pipeline in simulation.

## What Simulation Can Do

- Reveal whether claimed speed depends on assumed range.
- Show how camera motion, parallax, zoom, and frame timing can create apparent acceleration.
- Identify missing metadata preventing reconstruction.
- Compare scenario assumptions across repeated runs.
- Produce a ledger that separates inputs, estimates, uncertainty, and boundary language.

## What Simulation Cannot Do

- Replace calibrated evidence.
- Prove a real event happened.
- Establish origin, intent, or propulsion.
- Repair missing custody, timing, range, or sensor metadata.
- Convert unresolved into exotic.

## Simulation Report Rule

Simulation reports must list assumptions clearly. They must identify which outputs are range-dependent, which metadata is missing, and which claims remain unsupported.
