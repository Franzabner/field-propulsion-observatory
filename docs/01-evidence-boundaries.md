# Evidence Boundaries

Evidence levels must remain explicit. A more dramatic observation does not become stronger evidence unless the sensor chain, geometry, timing, and reconstruction become stronger.

No propulsion conclusion should be made from video alone.

## Evidence Levels

| Level | Definition | Public analysis posture |
| --- | --- | --- |
| Anecdotal report | A witness statement without reviewable instrument data. | Capture the claim, context, and uncertainty. Do not reconstruct trajectory. |
| Image/video only | Visual media without sufficient metadata. | Review apparent motion, compression, frame timing, and field-of-view limits. |
| Single-sensor event | One sensor stream with partial metadata. | Estimate only what the sensor and metadata can support. |
| Multi-sensor event | More than one sensor source, not necessarily calibrated. | Compare timing, line of sight, and correlation limits. |
| Calibrated multi-sensor event | Multiple sensors with known calibration, timing, and platform state. | Support bounded reconstruction if geometry is sufficient. |
| Independently reconstructable event | Evidence package allows another reviewer to reproduce estimates. | Publish assumptions, methods, uncertainty, and review notes. |
| Scientifically reviewable event | Data quality, provenance, calibration, and methods support formal technical review. | Permit stronger claims only within stated confidence limits. |

## Boundary Tests

- Is range measured, bounded, or assumed?
- Is platform motion known?
- Are timestamps synchronized?
- Is field of view known?
- Is the object tracked across frames with consistent identity?
- Are radar, infrared, and optical records independently correlated?
- Are artifacts, compression, glare, and parallax screened?

## Result Language

Allowed results include insufficient data, low confidence reconstruction, bounded reconstruction, multi-sensor supported, independently reconstructable, scientifically reviewable, explainable under known causes, and unresolved.

Unresolved is not a shortcut to exotic propulsion.
