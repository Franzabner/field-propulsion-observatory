# Review Checklist

## Public-Safety Review

- [ ] No aliens-confirmed claim.
- [ ] No anti-gravity-proven claim.
- [ ] No zero-point-propulsion-proven claim.
- [ ] No recovered craft or recovered material claim.
- [ ] No free energy claim.
- [ ] No government coverup framing.
- [ ] No classified-program claim.
- [ ] No real classified case conclusion.
- [ ] No real case accusation.
- [ ] No named disputed event used as evidence.
- [ ] No named real pilot, whistleblower, agency personnel, or sensitive source used as a case authority.
- [ ] No private DGX/rack topology.
- [ ] No sealed YOSO-YAi LLC IP.
- [ ] Synthetic examples are clearly fictional.
- [ ] No fake dataset, model, report, or Space release claim.
- [ ] Hugging Face status remains planned-only unless a reviewed artifact exists.

## Engineering Review

- [ ] Schemas validate.
- [ ] Examples validate against schemas.
- [ ] Scenario files validate against `schemas/simulation-scenario.schema.json`.
- [ ] Simulation scenarios run.
- [ ] Simulation ledgers do not output exotic propulsion conclusions.
- [ ] Generated ledgers remain ignored unless intentionally promoted as synthetic fixtures.
- [ ] Uncertainty language is explicit.
- [ ] Missing metadata is reported.

## Publication Review

- [ ] License selected.
- [ ] GitHub repo creation approved.
- [ ] Hugging Face remains no-action until a dedicated token/auth session.
- [ ] Upwork material remains draft-only unless separately approved.
- [ ] README status labels match `STATUS.md`.
