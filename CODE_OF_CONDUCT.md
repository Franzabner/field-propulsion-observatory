# Code of Conduct

This repository is for calm technical analysis. Contributors should use precise language, separate evidence from interpretation, and avoid sensational claims.

## Expected Conduct

- Treat unresolved questions as engineering problems, not identity arguments.
- Challenge claims by asking what data, metadata, and reconstruction are required.
- Keep public language respectful, specific, and reviewable.
- Preserve public/private/sealed boundaries.

## Unacceptable Conduct

- Harassment or personal attacks.
- Sensational, conspiratorial, spiritual, or science-fiction framing.
- Claims of confirmed exotic propulsion without calibrated, reviewable evidence.
- Attempts to publish secrets, tokens, private infrastructure, sealed IP, or real classified material.
