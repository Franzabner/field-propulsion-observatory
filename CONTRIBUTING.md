# Contributing

This scaffold accepts only public-safe contributions that strengthen evidence discipline, metadata clarity, simulation transparency, or hypothesis boundaries.

## Contribution Rules

- Use synthetic examples unless a real public case has explicit human review approval.
- Declare assumptions in simulation inputs and reports.
- Preserve the distinction between observation, metadata, reconstruction, simulation, and conclusion.
- Do not add alien, anti-gravity, zero-point propulsion, recovered-material, classified-program, or coverup claims.
- Do not add private infrastructure details, sealed IP, secrets, tokens, or real classified case material.
- Keep Hugging Face work planned-only unless a dedicated release session authorizes it.
- Keep Upwork material draft-only unless publication is explicitly reviewed.

## Validation

Before proposing changes, run:

```bash
npm run validate
npm test
```
